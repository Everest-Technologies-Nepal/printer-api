Setting Up WiFi Printer

Follow these steps to set up your WiFi printer:

  1. Download the code from the provided link.
  2. Install Node.js from Node.js official website.
  3. Set up and run the server on startup:
    4. npm install pm2 -g
npm install pm2-windows-startup -g
pm2-startup install
pm2 start ecosystem.config.js
  4. Start ngrok:
    5. ngrok http 3003 --domain=proven-magical-goldfish.ngrok-free.app --log=stdout > ngrok.log &
  5. To stop ngrok server running in the background, run the following command in PowerShell:
    6. Stop-Process -Name ngrok -Force

This guide will help you set up and run your WiFi printer server easily. If you have any issues, feel free to ask for help!
